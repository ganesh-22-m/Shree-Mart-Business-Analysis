# 🛒 Shree Mart – Business Analysis Portfolio Project

## 📌 Project Overview

Shree Mart is an end-to-end Business Analysis portfolio project developed for a retail e-commerce platform. The project demonstrates the complete Business Analysis lifecycle, from requirement gathering to documentation, data analysis, dashboard development, and solution design.

---

## 🎯 Business Problem

Retail businesses require an organized system to manage products, customers, orders, inventory, and reporting. The objective was to analyze business requirements and design a structured solution that improves operational efficiency and decision-making.

---

## 🎯 Project Objectives

- Gather and analyze business requirements
- Prepare business documentation
- Design SQL database structure
- Create Power BI dashboards
- Design Figma UI prototype
- Prepare UAT documentation
